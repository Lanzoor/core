# core

frontend source code for <https://lanzoor.dev>.

## Download

Clone the repository using one of the following commands:

```sh
git clone https://github.com/Lanzoor/core.git
# or
gh repo clone Lanzoor/core
```

## Install

Enter the `core` directory and install the required dependencies.

> **Note:** This project was built with `pnpm` in mind. Using `pnpm` is recommended.

```sh
pnpm install
# or
npm install
```

## Build

Build the project with:

```sh
pnpm build
# or
npm run build
```

For development, start the watcher instead:

```sh
pnpm watch
# or
npm run watch
```

## Serve

Serve the built site locally with:

```sh
pnpm dlx serve public
# or
npx serve public
```

Then open the URL shown in the terminal, typically <http://localhost:3000>.
