The Nippon Initial Teaching Alphabet Project

The logo of the Nippon Initial Teaching Alphabet project

The Nippon Initial Teaching Alphabet Project is an indie project created by Hurota Tobuchou,
which aims to help Japanese children learn English pronunciation and phonetics via the Initial Teaching Alphabet ,
a writing system designed to make early English reading and pronunciation easier to understand for beginners.

As part of this project, I will also be contributing by making a website dedicated to the project. Currently, a basic structure of the website has been published on https://nita-website-pink.vercel.app .
If you have any feedback, or just want to contribute to the project, feel free to join the NITA Discord server !
