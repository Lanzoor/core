API Structure Update Notice

This announcement serves as a notice regarding several important changes to the API structure of this website.

Complete Separation

The API directories have been fully moved from core to core-api, a completely separate git repository.
This change was made to ensure that all API routes are now hosted on api.lanzoor.dev instead of lanzoor.dev/api.
A redirect from lanzoor.dev/api/* to api.lanzoor.dev/* has been implemented for compatibility.

However, starting on June 1st, 2026, lanzoor.dev/api/* will be officially deprecated and will no longer be supported. If you are currently using any of my API services, please make sure to update your routes before the deadline.
