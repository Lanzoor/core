Consistent Illion Notation (CIN)

The better and more consistent notation for big numbers.

Introduction

Ever since I was a kid, I was a huge fan of idle games and large numbers. I also noticed the Conway-Wechsler system was used to describe large numbers. This used to fascinate me, since the names for large numbers sounded like a mythical artifact.

However, as I grew up, I started noticing how messed up the Conway-Wechsler system is. One of the biggest issues for me is that the Conway-Wechsler system uses phonological adjustments, which means that prefixes can change their shape based on where it's attached to.

So, I'm very proud to present my own version & improvement of the Conway-Wechsler system. Introducing... the Consistent Illion Notation! CIN for short.

What's the goal of CIN?

CIN is supposed to make illion index part of large numbers more straight-forward to understand, while also being phonologically consistent and easy to pronounce.

The Illion Index

We start by converting the number into its illion index form and mantissa. Given a number in scientific notation: \[ m \times 10^{e} \] Then, define the constant \(c\) as:

  • \(c = 3\) for Short Scale
  • \(c = 6\) for Long Scale

We can obtain the illion index \(n\) of this number with: \[ n = \left\lfloor \frac{e}{c} \right\rfloor \] We can obtain the remainder for the mantissa \(r\) with: \[ r = e \mod{c} \] So, the number would be expressed as: \[ m \times 10^{r} \quad n\text{-th illion} \]

CIN's goal is to help simplify the illion index part of big numbers, hence the name. You can use the steps below in order to convert the illion index into a CIN notation.

The Rule of Three

The Rule of Three (often abbreviated as TRoT) is a fundamental principle in CIN for reading and naming large illion numbers.
It states that every three digits of the illion index are grouped together and assigned a specific name, starting from the right.

For example, suppose we have a number with the illion index \(123,456\). The groups of this number would be \(123\) and \(456\). Refer to the table below, which explains how groups are named.

Examples

The number that has the illion index \(123,456\) would be broken down into:

  • nillion part - \(456\)
  • millinillion part - \(123\)

The number that has the illion index \(12,340,567\) would be broken down into:

  • nillion part - \(567\)
  • millinillion part - \(340\)
  • micrillion part - \(12\)
