Proto-Lanzavel

Welcome to my conlang! This is Proto-Lanzavel, which is a more ancient version of the modern Lanzavel language.
This is not a document for those who wants to study the language. It's a basic foundation of the conlang.
Once this language gets out of the development status, it will be adjusted to new learners.
But for now, there may be mistakes, inconsistencies, and variable information.

1. Getting Started

1-1. Purpose and Design Principles

Lanzavel is supposed to be a root language of all the future languages, so it can be referenced and evolved at the same time.
The conlang should be kept minimal and consistent, but it should also be able to properly convey meaning just like the other natural languages.
It's similar in structure as of in Japanese since it's a pro-drop and SOV language, but uses the Latin script as well as some accent markers.

1-2. Phonetics and Phonology

The first character of all alphabet names are capitalized. They help differentiate from the existing words.
Vowel names: [V] + ref
Consonant names: [C] + ef

1-2-1. Vowels

In PLZ, vowels are split into two types; "normal" vowels and "distorted" vowels.
Normal vowels have a more straight sound, while distorted vowels have a more distorted, alternative sound.
