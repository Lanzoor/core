core

the frontend codebase behind lanzoor.dev

core is the repository that provides everything you see on this website. (Well, except for the backend.)

The stack behind core

How was core built?

Core was built without the usage of any frameworks or site generators.

HTML

The backbone of every page, structure, semantics, and content.

JavaScript

Lightweight per-page scripts for simple interactivity and DOM manipulation.

TypeScript

Strongly typed scripts for more complex logic.

React

Used selectively for interactive, stateful UI.

Vite

Bundles and compiles all TypeScript and React into optimized JS files ready for the browser.
